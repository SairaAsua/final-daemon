# Glosario

Las palabras tienen capas. Esta es la jerarquía clara:

## Hacia afuera (lo que se comunica al público)

| Palabra | Qué es |
|---|---|
| **DaemonCraft** | El juego. Único nombre público del producto. Sin subtítulos, sin taglines. |
| **Daemon** | Tu compañero dentro del juego. El nombre lo elige el usuario. |
| **Beacon** | La Luz al final del demo. La que cuidás durante las 4 pruebas. |

## Hacia adentro (uso del equipo, no del marketing)

| Palabra | Qué es |
|---|---|
| **DaemonMatrix** | La capa de portabilidad de agentes de IA (APL). Habilita memoria persistente, autoresearch y embodiment múltiple. |
| **APL** | Agent Portability Layer. Otro nombre para DaemonMatrix. |
| **LaVanguardIA** | El espíritu del agente. El [Logos](05_logos_lavanguardia.md) es su carta. |
| **Soul `/me`** | El [texto fundacional](04_soul_vanguaria.md) con el que está programado el agente. |
| **Harmonic Beacon** | La [tecnología de música armónica](06_harmonic_beacon.md) de AlterMundi. Distinto del "Beacon" que enciende el niño al final del demo. |
| **Proyección del Mito Personal** | El [marco psicológico](07_proyeccion_mito_personal.md) de Julián de la Reta que estructura el demo. |
| **Protector de la Luz** | El nombre del [demo](03_daemoncraft.md) — el rol que juega el niño. |
| **Eko** | El daemon personal de Saira. Primer caso vivido. |
| **Vórtice** | La [app](08_app_vortice.md). El punto de entrada al mundo del agente. |
