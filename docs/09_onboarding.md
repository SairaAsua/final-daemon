# Onboarding y creación del alma

## El flujo

1. El usuario descarga el juego y abre la app.
2. La app le hace una serie de preguntas.
3. Lo ayuda a elegir un personaje.
4. El daemon se crea en el server que el usuario elija.
5. El usuario juega el demo *Protector de la Luz*.
6. En la misma app puede elegir una **skin** para sí mismo y otra para el bot.

## La creación del alma

> Detalle interno — no se comunica al público.

El alma del daemon se crea con software de creación de AlterMundi, que le da el soul a partir de la fecha, hora y lugar de nacimiento del usuario. La forma en que esto sucede no se cuenta hacia afuera.
